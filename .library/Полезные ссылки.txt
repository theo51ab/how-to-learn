https://emmet.io/
EMMET Документация по встроенной в VSCode автоподстановке по сокращениям+TAB, например .header<TAB> создает <div class="header"></div> или в CSS сокращение fz<TAB> или bgi<TAB> создаст соответственно font-size: ; и background-image: url();

Пользовательские сокращения (сниппеты) настраиваются через Файл > Настройки > Пользовательские фрагменты кода (либо по кнопке Шестеренка): для конкретного языка, для всех языков но конкретного проекта, либо глобальные. Подробнее: с 0:32:27 по 0:36:01 https://youtu.be/nxCLXMBl4e4

Auto Complete Tag
by Jun Han
Расширение VSCode автоматически вставляющее закрывающий тег. Такое умеет и сам VSCode, поэтому интереснее, что вместе с Auto Complete Tag устанавливается еще одно расширение - Auto Rename Tag: меняя название открывающего тега, меняется и закрывающий.

Project Manager

Создает новый блок в структуре навигации интерфейса для быстрого переключения между отдельными проектами.

BEM Helper
Box-Of-Hats
Нажатием заданной комбинации клавиш вставляет элемент (alt+b, e) или модификатор (alt+b, m) по методологии БЭМ, в т.ч. с учетом родителя. Плагин также может формировать файл стилей на основе БЭМ-разметки в файле HTML.

eCSStractor for VSCode
diz
В настройках по умолчанию - запросом ПКМ > eCSStractor Run копирует в буфер выдержку классов для вставки в CSS. Настройкой горячих клавиш можно определить команду eCSStractor Run (With BEM Nesting).

CSS Navigation
pucelle
Добавляет в подсказки набора HTML имеющиеся в CSS классы. Также нажатием F12 переносит из класса в HTML с установленным курсором к этому классу в файле, даже если файл CSS закрыт. А при нажатии alt+F12 появляется окно быстрого перебора классов для замены наименования, фактически не выходя из HTML.

Image prewiew
Kiss Tamas
Отображает миниатюру изображения сбоку окна редактирования, а при наведении - информацию об изображении.

Vscode Google Translate
funkyremi
Позволяет быстро переводить выделенный фрагмент текста. Возможно настроить комбинацию горячих клавиш активации плагина по умолчанию или для выбранного языка (Шестеренка > Сочетания горячих клавиш > [команда из окна настройки плагина]) и конкретный язык перевода (Шестеренка > Параметры > Расширения > Vscode Google Translate > [Указать предпочтительный язык перевода, например EN]).

Bracket Pair Colorizer 2
CoenraadS
Раскрашивает все скобки разными цветами, а также выделяет фрагмент в пределах открывающей и закрывающей скобки.
