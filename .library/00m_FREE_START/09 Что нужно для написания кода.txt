Редактор кода
Для начала вам понадобятся только браузер и редактор кода. Браузер у вас уже есть, а редактор нужно скачать и установить. Мы рекомендуем VS Studio Code — он бесплатный и распространённый.
Скачать редактор можно здесь: https://code.visualstudio.com/

Плагин для переноса кода из тренажёра в Codepen
Плагин для Google Chrome и Яндекс Браузера позволяет скачивать код из тренажёра Практикума и экспортировать его в Codepen.
Это может пригодиться для:
Локальной проверки работы кода
Редактирования кода, когда упражнение уже закрыто
Сохранения кода «на память»
Отправки сообщений об ошибках в Learning Support
Установка:
Скачайте архив.
Распакуйте его.
Откройте раздел расширений браузера:
В Google Chrome вставьте в адресную строку: chrome://extensions/
В Яндекс Браузере: browser://extensions/
Включите режим разработчика.
Нажмите «Загрузить распакованное расширение». Выберите папку с распакованным расширением загрузите.
Использование:
Откройте тренажёр с кодом. При клике на иконку треугольника скачивается только код этой вкладки.
При нажатии на иконку Практикума в правом верхнем углу и ссылку «Download all tabs» – скачивается архив с кодом из всех вкладок.
При нажатии на иконку Практикума и ссылку «Share with codepen» – автоматически создаётся вкладка с Codepen с кодом из тренажёра.
