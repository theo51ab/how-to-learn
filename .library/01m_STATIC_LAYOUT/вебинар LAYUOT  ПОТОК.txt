СПЕЦИФИЧНЫЙ ПОТОК

display: flex; - flex меняет контекст потока, по умолчанию меняет направление на слева-направо (блоки в гориз ряд) или иначе через flex-direction: revert; flex-direction: row; flex-direction: row-reverse; (справа-налево) и др.

свойство стиля column-count: 3; - автоматом разбивает текст на 3 колонки

display: grid; в отл от flex (направление потока, выравнивание отступов - в общем выравнивание) распределяет контент по сетке не используя таблицы (которые след использ только для таблиц). И flex и grid удобно использовать при адаптиве под разные платформы, в т.ч. мобильный формат
display: grid;
grid-template-columns: repeat(6, auto);
...

position: fixed; - вырывает элемент из потока - при скролле остается на месте
position: absolute; - вырывается из потока, при этом абсолютное положение относительно родительского элемента
position: relative;
position: sticky; - прилипающий элемент как шапка сайта, которая остается на экране при скроллинге
